RetroComputer Words API

V1 | maintained by github.com/abulerforrest

This is a simple REST API containing a huge collection of Retro Computer Words (14476 words) which I've gathered from text of various old 80's Atari magazines, examples: Antic, Compute!, Analog Computing Video And Arcade Games, Hi-Res Magazine, ROM Magazine etc.
(Source: https://www.atarimagazines.com) I've also crawled words from video game lists such as classic Nintendo, Atari games and from Wikipedia pages regarding computer technology/science.

So, why the effort setting up this API?

I'm currently working on a game called Word Razer which is a retro style word game that will use this API in order to serve its random words in the game.

API Endpoints

[GET]
api/v1/words/all
→ GET ALL WORDS
[GET]
api/v1//words/limit/:limit
→ GET WORDS BY LIMIT [number 1-14476]
[GET]
api/v1//words/random/all
→ GET ALL RANDOM WORDS
[GET]
api/v1//words/random/:limit
→ GET RANDOM WORDS BY LIMIT [number 1-14476]
[GET]
api/v1/words/id/:id
→ GET A SPECIFIC WORD BY OBJECT_ID
Returns the result of word objects in JSON-format as "word": [{ "_id":"{OBJECT_ID}","word":"{WORD}" }]"
