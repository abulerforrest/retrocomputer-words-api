RetroComputer Words API

V1 | maintained by github.com/abulerforrest

Description

This is a simple REST API containing a big collection of Retro Computer Words (14476 words) which I've gathered from text of various old 80's computer magazines, examples: Antic, Compute! Analog Computing Video And Arcade Games Hi-Res Magazine (source: https://www.atarimagazines.com) and more. I also gathered text from classic Nintendo and Atari games lists as as well from Wikipedia pages regarding computer technology.

Endpoints

[GET] api/v1/words/all → GET ALL WORDS

[GET] api/v1//words/limit/:limit → GET WORDS BY LIMIT [number 1-14476]

[GET] api/v1//words/random/all → GET ALL WORDS BY RANDOM

[GET] api/v1//words/random/:limit → GET RANDOM WORDS BY LIMIT [number 1-14476]


Retruns the selected number of JSON word objects "word": [{ "_id":"{OBJECT_ID}","word":"{WORD}" }]"
